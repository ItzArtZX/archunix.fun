Requirements:

- Root Privileges

Since the script modifies system files (/etc/os-release, /usr/lib/os-release) and /usr/share/pixmaps, root (sudo) access is required.

- Arch Linux

Works best on Arch Linux or Arch-based distros (but could be adapted for others).

- wget or curl

- coreutils

- sed (part of coreutils but sometimes installed separately)
